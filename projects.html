Barricane Projects

The projects below are all non-customer projects. They are either:

  • open-source projects which Barricane has developed
  • open-source projects to which Barricane staff have contributed code
  • in-house projects

BarricaneDB

BarricaneDB is a simple in-process persistence mechanism. It is developed and maintained as an open-source project by Barricane.

The source is available on GitHub.

mysql-native

To quote: "Yet another pure JS async driver. Pipelining, prepared statements."

You can see Barricane's latest contribution on GitHub.

corner

jQuery Corner Plugin (by Dave Methvin and Mike Alsup)

You can see Barricane's latest contribution on GitHub.

Mapstraction

Mapstraction is a library that provides a common API for various javascript mapping APIs to enable switching from one to another as smoothly as possible. Developers can code their applications once, and then easily switch mapping provider based on project needs, terms and conditions, and new functionality.

You can see Barricane's latest contribution on GitHub.

Tipslate

Tipslate is an idea for a micropayments service. It is currently in development and alpha testing. Have a look, if you like. It may break.
